# Genshin Auto DPS Analyzer

## 프로젝트 소개

원신 캐릭터 파티랑 행동 순서를 자동으로 계산해서
딜이 가장 잘 나오는 조합을 찾는 프로그램입니다.

웹에서 정보 모아서 정리하고,
gcsim 돌려서 실제 DPS 기준으로 결과를 뽑습니다.

---

## 기능

### 1. 데이터 수집

캐릭터별로 검색해서
파티, 무기, 성유물 정보 가져옵니다.

### 2. 전처리

gcsim에서 사용할 행동 조건을 만들고
필요한 데이터 정리합니다.

### 3. 엔진

파티 순서를 바꿔가면서
gcsim으로 딜 계산하고
가장 높은 결과를 선택합니다.

### 4. 새부분석

결과 정리하고
실패한 캐릭은 제외해서
최종 파일로 저장합니다.

---

## 설정값

### 1번 (수집)

* 동시 실행 수
* 검색 결과 개수
* 문서 저장 개수
* 최소 글자 수
* txt 저장 여부
* 제외 키워드
* 제외 사이트

### 2번 (전처리)

* 동시 실행 수

### 3번 (엔진)

* 동시 실행 수
* iteration
* duration
* timeout

### 4번 (새부분석)

* 동시 실행 수
* 개체 수
* 세대 수
* 토큰 범위

---

## 결과 파일

### data

* teams.json → 파티 정보
* gear.json → 장비 정보
* best_orders.json → 최고 순서

### output

* failed_runs.csv → 실패한 캐릭
* 최종 결과 파일
* 로그 및 config

---

## 핵심 방식

파티 하나 정해놓고
순서만 바꿔가면서 전부 돌린 다음
딜 가장 높은 걸 고르는 방식입니다.
