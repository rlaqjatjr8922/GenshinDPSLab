# GenshinDPSLab

원신 캐릭터 조합과 행동 로테이션을 자동 분석하여  
gcsim 기반으로 높은 DPS 조합을 탐색하는 Python GUI 프로그램입니다.

---

## 📌 프로젝트 소개

GenshinDPSLab는 원신 캐릭터의 파티 구성, 장비, 행동 순서를 자동으로 분석하여  
가장 높은 DPS를 도출하는 프로그램입니다.

데이터 수집 → 전처리 → 시뮬레이션 → 결과 분석의 전체 파이프라인을 자동화했으며,  
Tkinter 기반 GUI를 통해 단계별 실행과 진행 상황 확인이 가능합니다.

---

## 🔘 실행 단계 (Pipeline)

1. 캐릭터 데이터 수집  
   - 캐릭터, 무기, 성유물 등 기본 데이터 수집  
   - JSON 형태로 저장  

2. 규칙 데이터 수집  
   - 행동 가능 조건 및 액션 규칙 정리  
   - 시뮬레이션에 필요한 로직 데이터 생성  

3. DPS 시뮬레이션  
   - 파티 조합 및 행동 순서를 기반으로 DPS 계산  
   - gcsim을 이용한 실제 딜 시뮬레이션 수행  

4. DPS 정밀 분석  
   - 최고 DPS 결과 선별  
   - 로테이션 최적화 및 상세 결과 분석  

---

## 🖥️ GUI 기능

- 단계별 실행 버튼 (Pipeline 구조)
- 진행률 바 (단계별 / 전체 진행)
- 실시간 로그 출력
- 데이터 상태 확인 (캐릭터, 장비, 파티 등)
- 전체 자동 실행 기능
- 작업 중 중지 / 초기화 / 재로드 기능
