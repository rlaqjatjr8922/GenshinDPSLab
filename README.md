# GenshinDPSLab

원신 캐릭터 조합과 행동 로테이션을 자동 분석하여 gcsim 기반으로 높은 DPS 조합을 탐색하는 Python GUI 프로그램입니다.

---

## 📌 프로젝트 소개
GenshinDPSLab는 원신 캐릭터의 파티 구성, 장비, 행동 순서를 자동으로 분석하여 가장 높은 DPS를 도출하는 프로그램입니다.

데이터 수집 → 전처리 → 시뮬레이션 → 결과 분석의 전체 파이프라인을 자동화했으며, Tkinter 기반 GUI를 통해 단계별 실행과 진행 상황 확인이 가능합니다.

---

## ⚙️ 주요 기능

### 1. 데이터 수집
- 캐릭터, 무기, 성유물, 파티 정보 수집
- JSON 형태로 저장 및 관리

### 2. 전처리
- gcsim에서 사용할 데이터 형태로 변환
- 행동 조건 및 가능한 액션 정리

### 3. 시뮬레이션 엔진
- 파티 조합 및 행동 순서를 바꿔가며 DPS 계산
- gcsim CLI를 활용한 실제 딜 시뮬레이션 수행

### 4. 후처리
- 최고 DPS 결과 저장
- 실패 데이터 및 로그 정리

---

## 🖥️ GUI 기능
- 단계별 실행 버튼 (수집 → 전처리 → 엔진 → 후처리)
- 진행률 바 표시 (멀티 진행)
- 실시간 로그 출력
- 데이터 상태 확인 (캐릭터, 장비, 파티 등)
- 전체 자동 실행 기능
